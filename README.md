# razuvaev.tv

Personal music streaming page featuring YouTube music videos with integrated music service links.

## Features

- **Header**: Logo linking to razuvaev.me, tab navigation, and Telegram button
- **Tab System**: "On Repeat" and "AOTY" tabs for different playlists
- **YouTube Player**: Embedded YouTube player with custom overlay
- **Track Information**: Album art, track name, album name, and artist name
- **Music Service Links**: Spotify, Apple Music, and Yandex Music links
- **Navigation**: Previous/Next track buttons and play/pause functionality

## Structure

```
razuvaev.tv/
├── index.html          # Main HTML structure
├── styles.css          # Styling for the website
├── script.js           # JavaScript functionality
├── playlist.json       # Sample playlist data
├── README.md           # This file
└── package.json        # Node.js configuration
```

## Setup

1. Clone the repository
2. Open `index.html` in a web browser
3. The website will load the sample playlist and display the first track

## Customization

- Update `playlist.json` with your own music tracks
- Modify `styles.css` to change the visual design
- Update the logo link and Telegram button URL in `index.html`

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Responsive design for mobile and desktop
